# quiz-application
An interactive multiple choice quiz game with scoring built in Python
An interactive multiple choice quiz game built using
Python during my internship at SkillInfyTech IT Solutions.

## About
This project presents users with multiple choice 
questions one by one, takes their input and calculates
the final score at the end. Questions are stored in a 
pre-defined list inside the program.

## Features
- Multiple choice questions with 4 options each
- Tracks correct and wrong answers
- Displays final score at the end
- Input validation for invalid choices

## How to Run
Make sure Python is installed on your system.
python quiz.py

## Concepts Used
- Python Lists and Dictionaries
- Loops and Conditionals
- User Input and Output
- Exception Handling
- Score Tracking Logic
